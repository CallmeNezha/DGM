Direct Graphical Models C++ library
===================================

DGM is a C++ dynamic link library implementing various tasks in probabilistic 
graphical models of discrete data with pairwise (and unary) potentials. 
The library may be used for the Markov and Conditional Random Fields (MRF / CRF), 
as well as for Markov Chains ans Bayesian Networks.

Modules:
-------
* DGM - the main library
* FEX - feature extraction module
